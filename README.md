# vue-directive-waves
vue自定义指令(Vue.directive)实现点击水波纹效果

## 线上[demo](https://gassnake999.github.io/vue-directive-waves/)

![效果图](https://i.loli.net/2019/04/16/5cb5838b618db.gif)

1. 克隆项目
```
git clone https://github.com/gassnake999/vue_directive_waves.git
```
2. 构建项目
```
yarn
或者
npm install
```
3. 启动项目
```
yarn serve
或者
npm run serve
```
